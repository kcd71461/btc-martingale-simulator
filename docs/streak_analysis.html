라운드별 조건부 승률 — 실제 데이터 vs 이론(50%)

조건부 승률: 해당 라운드까지 도달했을 때 다음 캔들이 반전(승리)할 확률.
라운드 N = streak 길이 (4+N)번째 캔들이 반전하는지 여부.
이론(빨간 점선): 캔들이 독립적이고 각 방향 확률 = 50% 가정.

라운드별 도달 횟수 (Funnel)
